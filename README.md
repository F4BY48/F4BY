# F4BY
Proyecto de Estructura de Datos y Programacion Orientada a Objetos

## Descripcion
F4BY es una aplicacion de renta de coches, donde el usuario pueden:
- Registro de usuario
- ver todos los coches que ofrece la empresa
- buscar coche especifico con categoria / filtros
- seleccionar un coche y ver que horarios esta disponible
    - requisito: El coche necesita 30min de limpieza despues de ser usado
- El usuario puede realizar una peticion para rentar el coche
- Si la peticion es valida se le reservara el coche en el horario
- Se le ofrecen varios tipos de seguro, debe elegir uno

funcionalidades extra:
- Descuento Caliente: rentar coche a ultima hora
- Programa de puntos "Millas F4BY"
- Penalizacion por danos, multas, o entrega tardia
